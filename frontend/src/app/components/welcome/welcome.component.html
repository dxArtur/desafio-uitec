Sobre a aplicação

Uma aplicação responsiva (Experimente usar o seu navegador no modo mobile), desenvolvida em PHP (backend) e Angular (frontend) usando MySQL para persistência de dados para registrar suas transações financeiras e ver seu extrato com todas as suas transações realizadas com opção de filtragem, de uma forma simples e rápida.

Abaixo segue instruções para uso.

Cadastro de Transações

Navegar para a Página de Cadastro: Clique na opção "Cadastrar Novas Transações" na navbar.

Preencher o Formulário:

Ex: Compra de supermercado
Ex: 150.00

Salvar a Transação: Clique no botão abaixo para cadastrar a transação.

Visualização do Extrato

O extrato exibe uma lista de todas as transações cadastradas, permitindo que você acompanhe suas finanças:
